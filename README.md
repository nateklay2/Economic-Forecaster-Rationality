Project Highlights:

Rationality Analysis: Evaluated if professional forecasts align with real-world outcomes.

Hypothesis Testing: Conducted simulation-based testing by building 95% confidence intervals for subjective probability distributions

Data Integration: Merged individual forecaster data from the Philadelphia Fed with real-time unemployment rates from the FRED database

Data Engineering: Cleaned multi-index DataFrames (ID and Date), handled null values, and validated that probabilities summed to 100%
